# Stok Material Promo Management

Aplikasi internal untuk mengelola stok material promosi LPG PSO Kalbar.

## Stack
- **Frontend:** React 19 + Vite + Tailwind 4
- **Backend:** Express.js + better-sqlite3
- **Database:** SQLite
- **Auth:** JWT (HTTP-only cookie)
- **Domain:** stok.khatulistiwa.cloud
